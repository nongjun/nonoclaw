# 变更日志：Cursor 授权策略决策

**日期**：2026-03-18  
**决策人**：农总  

## 结论

- Cursor CLI **不支持**第三方 API Key 作为调用凭证；该方向不再投入验证。
- 平台采用：**多个 Cursor 账号 → 多个官方 User API Key → Key 池**，配合**欠费/额度类错误时自动轮询切换**，保障用户侧连续可用。
