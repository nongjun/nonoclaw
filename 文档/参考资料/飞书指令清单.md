# 飞书指令清单

龙虾军团飞书 Bot 支持的所有指令，在飞书对话中直接输入即可触发。

## 基础指令

| 指令 | 说明 |
|------|------|
| **/帮助** | 显示所有可用指令及简要说明 |
| **/状态** | 查看当前 Bot 运行状态（模型、记忆、会话信息等） |
| **/新对话** | 清空当前上下文，开启全新对话 |
| **/终止** | 终止当前正在执行的任务 |

## 会话管理

| 指令 | 说明 |
|------|------|
| **/会话** | 查看当前会话信息 |
| **/模型** | 查看或切换当前使用的 AI 模型 |
| **/密钥** | 查看密钥配置状态（不显示实际密钥值） |

## 记忆系统

| 指令 | 说明 |
|------|------|
| **/记忆** | 查看当前长期记忆内容 |
| **/记录** | 手动记录一条信息到长期记忆 |
| **/整理记忆** | 触发记忆整理，合并去重、归类摘要 |

## 任务与维护

| 指令 | 说明 |
|------|------|
| **/任务** | 查看定时任务列表及执行状态 |
| **/心跳** | 手动触发一次心跳检查（健康检查 + 待办提醒） |

## 使用说明

- 所有指令均以 `/` 开头，不区分大小写
- 指令可在私聊或群聊中使用（群聊中需 @Bot）
- 部分指令支持子命令，输入主指令后会提示可用的子命令
