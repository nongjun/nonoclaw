# 环境变量说明

项目运行所需的环境变量配置，参考 `.env.example` 文件。将 `.env.example` 复制为 `.env` 并填入实际值即可。

## 必填项

| 变量名 | 说明 |
|--------|------|
| **CURSOR_API_KEY** | Cursor Agent CLI 的 API 密钥，用于调用 Cursor AI 能力 |
| **FEISHU_APP_ID** | 飞书应用的 App ID，用于飞书 Bot 长连接模式 |
| **FEISHU_APP_SECRET** | 飞书应用的 App Secret，与 App ID 配对使用 |

## 可选项

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| **CURSOR_MODEL** | `opus-4.6-thinking` | AI 模型选择，可选 `opus-4.6-thinking`、`sonnet-4`、`auto` 等 |
| **VOLC_STT_APP_ID** | — | 火山引擎语音识别的应用 ID，启用语音转文字功能时需配置 |
| **VOLC_STT_ACCESS_TOKEN** | — | 火山引擎语音识别的访问令牌，与 STT App ID 配对使用 |
| **VOLC_EMBEDDING_API_KEY** | — | 火山引擎向量嵌入的 API 密钥，启用记忆搜索功能时需配置 |
| **VOLC_EMBEDDING_MODEL** | `doubao-embedding-vision-250615` | 火山引擎向量嵌入模型名称 |

## 安全提醒

- `.env` 文件已在 `.gitignore` 中，**绝不提交到 Git 仓库**
- 生产环境的密钥应与开发环境不同
- 发现密钥泄露时必须立即轮换
