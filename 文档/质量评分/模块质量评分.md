# 模块质量评分

| 模块 | 成熟度 | 测试覆盖 | 改动风险 | 备注 |
|------|--------|---------|---------|------|
| server.ts（主服务） | ⭐⭐⭐⭐ | 低 | 高 | 2500 行单文件，核心主链路，改动需谨慎 |
| gateway.ts（网关） | ⭐⭐⭐⭐ | 低 | 中 | 飞书连接稳定层，极少重启 |
| feishu/（飞书集成） | ⭐⭐⭐ | 低 | 中 | 17 个文件，功能完整，缺少单元测试 |
| memory.ts（记忆系统） | ⭐⭐⭐⭐ | 低 | 中 | SQLite + 向量 + FTS5，设计成熟 |
| memory-tool.ts（记忆 CLI） | ⭐⭐⭐ | 低 | 低 | 独立 CLI，接口简洁 |
| scheduler.ts（调度器） | ⭐⭐⭐ | 低 | 中 | 支持三种调度模式 |
| heartbeat.ts（心跳） | ⭐⭐⭐ | 低 | 低 | 模块稳定，逻辑简单 |
| bridge.ts（API 桥接） | ⭐⭐ | 低 | 低 | OpenAI 兼容层，使用较少 |
| templates/（模板系统） | ⭐⭐⭐⭐ | — | 低 | 14 规则 + 9 数据模板 + 1 技能 |
| .cursor/（项目配置） | ⭐⭐⭐ | — | 低 | 5 Agent + 21 命令 + 11 技能 + 23 规则 |
| plugins/turix-cua/ | ⭐⭐ | 低 | 低 | Python 桌面操控，独立进程 |

### 评分标准

- ⭐ — 原型阶段，随时可能重写
- ⭐⭐ — 基本可用，缺少测试和文档
- ⭐⭐⭐ — 生产可用，有基本测试
- ⭐⭐⭐⭐ — 稳定运行，文档完善
- ⭐⭐⭐⭐⭐ — 成熟稳定，经过充分验证

### 改动风险说明

- **低风险**：模块稳定，改动不影响其他模块
- **中风险**：基本稳定，改动需测试验证
- **高风险**：核心链路，改动需非常谨慎

### 重点关注

1. **server.ts 体量过大**（2500 行），建议后续拆分
2. **整体测试覆盖率极低**，所有模块均缺少单元测试
