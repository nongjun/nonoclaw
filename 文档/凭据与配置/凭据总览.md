# 龙虾军团 — 凭据与配置总览

> ⚠️ 仅用于开发测试环境，生产环境另行管理

## Cursor Agent CLI

| 项目 | 值 | 说明 |
|------|-----|------|
| API Key | （见 .env） | [Cursor Dashboard](https://cursor.com/dashboard) → Integrations → User API Keys |
| 默认模型 | claude-4.6-opus-high-thinking | 可通过飞书 /模型 指令切换，`agent models` 查看可用列表 |

## 飞书机器人

| 项目 | 值 | 说明 |
|------|-----|------|
| App ID | （见 .env） | [飞书开放平台](https://open.feishu.cn) 创建企业自建应用 |
| App Secret | （见 .env） | 同上 |
| 连接方式 | WebSocket 长连接 | 无需公网地址 |
| 权限 | im:message, im:message.group_at_msg, im:resource | |
| 事件订阅 | im.message.receive_v1 | 长连接模式 |

## 火山引擎（可选）

### 语音识别（豆包大模型 STT）

| 项目 | 值 | 说明 |
|------|-----|------|
| App ID | （见 .env VOLC_STT_APP_ID） | [火山引擎控制台](https://console.volcengine.com/speech/app) |
| Access Token | （见 .env VOLC_STT_ACCESS_TOKEN） | 同上 |
| 资源 ID | volc.bigasr.sauc.duration | 需开通「大模型流式语音识别」 |
| 协议 | WebSocket 二进制 | wss://openspeech.bytedance.com/api/v3/sauc/bigmodel_nostream |

### 向量嵌入

| 项目 | 值 | 说明 |
|------|-----|------|
| API Key | （见 .env VOLC_EMBEDDING_API_KEY） | 火山引擎 API Key |
| 模型 | doubao-embedding-vision-250615 | 默认值 |
| Endpoint | https://ark.cn-beijing.volces.com/api/v3/embeddings/multimodal | |

## 环境变量文件位置

- `.env` — 主配置文件（热更换，修改后 2 秒内自动生效）
- `.env.example` — 模板文件
- `projects.json` — 项目路由配置（上层目录，热更换）

## 本地依赖

| 依赖 | 路径/安装方式 | 说明 |
|------|-------------|------|
| Bun | brew install bun | 运行时 |
| Cursor Agent CLI | ~/.local/bin/agent | Cursor IDE 安装后自带 |
| ffmpeg | brew install ffmpeg | 语音转码（可选） |
| whisper-cpp | brew install whisper-cpp | 本地语音识别兜底（可选） |
