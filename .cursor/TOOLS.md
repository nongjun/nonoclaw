## Cursor Agent 原生能力

| 能力 | 说明 |

## 记忆系统

| 语义搜索 | `bun memory-tool.ts search "关键词"` （如已部署）|
| 直接读取 | 读 `.cursor/MEMORY.md` + `.cursor/memory/*.md` |
| 写入记忆 | 编辑 `.cursor/MEMORY.md` 或日记文件 |

## IM 桥接

指令来自 IM 消息（飞书/钉钉/企微等，取决于配置）。
- 语音 → 自动转文字
- 图片/文件 → 下载到本地路径
- 回复 → 自动推送给用户

### 消息卡片限制

- JSON 最大 30KB（约 3500 中文字）
- 单卡最多 5 个表格
- 超长内容写文件，回复给摘要
