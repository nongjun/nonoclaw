## 质量评分模板

```markdown
# 模块质量评分

> 最后更新: [日期]

| 模块 | 成熟度 | 测试覆盖 | 文档完整度 | 改动风险 | 备注 |
|------|--------|---------|-----------|---------|------|
| | ⭐⭐⭐⭐⭐ | 高/中/低 | 高/中/低 | 低/中/高 | |

### 评分标准

- ⭐ — 原型阶段，随时可能重写
- ⭐⭐ — 基本可用，缺少测试和文档
- ⭐⭐⭐ — 生产可用，有基本测试
- ⭐⭐⭐⭐ — 稳定运行，测试和文档完善
- ⭐⭐⭐⭐⭐ — 成熟稳定，经过充分验证

### 改动风险说明

- **低风险**：模块稳定，测试完善，放心改动
- **中风险**：基本稳定，改动需测试验证
- **高风险**：模块脆弱，改动需非常谨慎，建议先补测试
```
