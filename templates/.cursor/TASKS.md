# TASKS.md - 定时任务

> 你可以通过 IM 告诉 AI 设定定时任务，AI 会自动创建。

## 当前任务

暂无。在 IM 中发送类似以下消息即可创建：

- "每天早上9点检查邮件"
- "每小时检查服务器状态"  
- "明天下午3点提醒我开会"

## 管理

| 操作 | 方法 |
|------|------|
| 查看任务 | 告诉 AI"查看定时任务" |
| 暂停任务 | 告诉 AI"暂停任务 <ID>" |
| 删除任务 | 告诉 AI"删除任务 <ID>" |
| 手动触发 | 告诉 AI"立即执行任务 <ID>" |

任务数据存储在工作区根目录的 `cron-jobs.json` 中。
