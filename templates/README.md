# 工作区模板

这是龙虾军团（nonoclaw）工作区初始化模板。每个新工作区从这里复制，获得独立的核心闭环体系。

## 包含内容

### 身份与人格
| 文件 | 用途 |
|------|------|
| `.cursor/SOUL.md` | AI 灵魂、人格、核心准则 |
| `.cursor/IDENTITY.md` | 名字、气质、Emoji |
| `.cursor/USER.md` | 用户画像 |
| `.cursor/BOOTSTRAP.md` | 出生仪式（首次启动后删除）|

### 运行系统
| 文件 | 用途 |
|------|------|
| `.cursor/MEMORY.md` | 长期记忆骨架 |
| `.cursor/HEARTBEAT.md` | 心跳检查清单 |
| `.cursor/BOOT.md` | 启动自检 |
| `.cursor/TASKS.md` | 定时任务文档 |
| `.cursor/TOOLS.md` | 能力清单 |

### 规则（每次会话自动加载）
| 文件 | 用途 |
|------|------|
| `soul.mdc` | 人格注入 |
| `agent-identity.mdc` | 身份 + 输出限制 |
| `user-context.mdc` | 用户上下文 |
| `workspace-rules.mdc` | 安全边界 |
| `memory-protocol.mdc` | 记忆协议 |
| `heartbeat-protocol.mdc` | 心跳协议 |
| `scheduler-protocol.mdc` | 调度协议 |
| `tools.mdc` | 工具能力 |
| `cursor-capabilities.mdc` | 能力决策树 |
| `基础规则.mdc` | 全局基础约束 |
| `安全红线.mdc` | 安全底线 |
| `文档策略.mdc` | 文档极简主义 |
| `核心体系保护.mdc` | 核心体系保护 |
| `通用编码风格.mdc` | 编码风格 |

## 使用方式

初始化新工作区时，将整个 templates 目录内容复制到目标工作区即可。
